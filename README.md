# 📚 Tuition Fee Tracker PWA

Track your children's tuition fees, classes, payments and budget.

## Features
- Multiple children support
- Monthly / Per Session / Termly fee types
- Overdue payment alerts
- Budget tracking with 6-month trend chart
- Full payment history & logs
- Sri Lankan Rupee (₨) + 6 other currencies
- Export/Import JSON backup
- Works 100% offline

## Deploy to GitHub Pages (Free Hosting)

1. Create a new GitHub repository (e.g. `tuition-tracker`)
2. Upload all files from this zip: `index.html`, `manifest.json`, `sw.js`
3. Go to **Settings → Pages → Source → main branch → / (root) → Save**
4. Your app will be live at: `https://YOUR-USERNAME.github.io/tuition-tracker`

## Install on Android (Chrome)
1. Open the GitHub Pages URL in Chrome
2. Tap the **three-dot menu (⋮)** → **"Add to Home Screen"**
3. Tap **Install** — app icon appears on your home screen
4. Works fully offline after first load

## Install on iPhone (Safari)
1. Open the GitHub Pages URL in Safari
2. Tap the **Share button** → **"Add to Home Screen"**
3. Tap **Add**

## Data Storage
- Data saved in **Chrome localStorage** on your device
- Persists indefinitely — survives for years
- Use ⚙️ → Download Backup to export .json file
- Restore from any previous backup anytime
